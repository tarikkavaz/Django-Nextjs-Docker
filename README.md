# Django & Next.js in Docker
## Yet another Boilerplate 


### Development Setup:

```bash
git clone https://github.com/tarikkavaz/Django-Nextjs-Docker.git
cd Django-Nextjs-Docker
yarn 
yarn setup
```

### Run Development:

```bash
yarn dev
```
